Rules/Groups bearbeiten

Möchte ein Projektmitglied eine Rule/Group bearbeiten, muss er auf der Webseite korrekt eingeloggt sein 
und in seinem Project die Rule/Group anwählen. Der registrierte User kann den Bearbeiten-Button anklicken 
und gelangt somit in den Bearbeitungsmodus. 
Regelbearbeitung:  Man kann ID, Severity, Description und Cypher direkt im Textfeld bearbeiten. 
Geht man ins Textfeld von ID und Cypher, erscheint ein Popup, welches auf die Verweise und möglichen  
Beeinträchtigung anderer Groups hinweist. Bei Bestätigung kann man den Text editieren, ansonsten wird der 
Fokus auf den Bearbeiten-Button gelegt. Der registrierte User kann den Bearbeitungsvorgang abbrechen, indem er 
auf den Abbrechen-Button klickt.  Danach gelangt er wieder zur Übersicht der Rules.
Gruppenbearbeitung:  Man kann die ID verändern oder Rules und (von der Aktuellen verschiedene-) internal Group 
aus der Group entfernen. Der registrierte User kann den Bearbeitungsvorgang abbrechen, indem er auf den Abbrechen-Button klickt. 
Danach gelangt er wieder zur Übersicht der Groups.
