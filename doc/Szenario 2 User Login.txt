User Login

Der User gibt auf der Startseite im Loginbereich seinen Usernamen und sein Passwort ein.
Nach einem Klick auf den Bestätigen-Button wird der Username in der User Datenbank gesucht.
Ist dieser nicht vorhanden, wird die Meldung "Der User exisitert nicht" ausgegeben.
Wenn der Username existiert, wird das vom User eingegebene Passwort mit dem in der Datenbank hinterlegten abgeglichen.
Stimmen diese nicht überein wird die Meldung "Passwort nicht korrent, bitte erneut versuchen." ausgegeben.
Ist der Abgleich korrekt bekommt der User den Status registrierter User
und damit Zugriff auf sein Profil und die damit verknüpften Inhalte.
