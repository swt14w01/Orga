5.1)
Gruppe hinzufügen

Der User möchte eine Gruppe hinzufügen. Dazu befindet er sich auf der Seite "Gruppe erstellen".
Er betätigt einen Button, welcher eine neue Ansicht öffnet. Hier wird der User aufgefordert, spezifische Angaben
bzgl. der neuen Gruppe zu machen wie beispielsweise ID, Tags.
Das System überprüft die Eingaben auf ihre Gültigkeit. Ist die ID bereits vergeben, so wird der User dazu
Wenn die Eingaben valide sind, wird ein neuer Eintrag in der Regeldatenbank mit den entsprechenden Eingaben
getätigt.
